## Testing Data Used
Two datasets used to train this AI.
Both are required to be in the directories listed below to properly train and test the model.

**1. Human and AI Separate Dataset**

Contains:
- 4755 human written examples (CodeNet)
- 2828 ai generated examples (CodeStral, Gemini, CodeLLaMA)
- Languages: Python only

This set was given by our instructor.\
Students can find it in canvas or click the link below.

```
!!! DEVELOPERS !!!
For local development, these filepaths are required.
/data/raw/human/human_selected_dataset.csv
/data/raw/human/created_dataset_with_llms.csv
```

[Dataset Download](https://huggingface.co/datasets/basakdemirok/AIGCodeSet)

**2. Human and AI Merged Dataset**

Contains:
- 5000 human written examples (CodeNet)
- 5000 ai generated examples (ChatGPT ver. 3.5 and 4)
- Languages: C, C++, Python, Java

This set was found online link provided below.

```
!!! DEVELOPERS !!!
For local development, this filepath is required
/data/raw/human_and_ai/HumanVSAI_CodeDataset.csv
```
[Article Link](https://www.sciencedirect.com/science/article/pii/S2352340926000806) and [Dataset Download](https://data.mendeley.com/datasets/kjh95n54f8/1)

## Directory Structure
```
ai/
│
├── data/ # all data related stuff
│   ├── metadata/   # stuff generated by actual predictions only
│   ├── processed/  # stuff generated by preprocess.py only
│   ├── predict/      # can put other data related for test_prediction.py
│   └── raw/
│       ├── ai/     # ai only datasets
│       ├── human/      # human only datasets
│       └── human_and_ai/       # merged datasets
│
├── models/     # saved model information post training and testing
│
└── src/    # code
```

# HOW TO USE - PROCESS
1. `preprocess.py` – Prepare and clean data  
2. `train.py` – Train the model  
3. `test.py` – Evaluate model performance  
4. `test_prediction.py` – Run predictions on other data
   - temporary file for testing the service
