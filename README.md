# CodeHawk-Automated Grading System
CodeHawk is a secure, web-based automated grading system designed to support programming-based courses. It streamlines assignment management, grading, plagiarism detection, and reporting for **students, faculty, and teaching assistants (TAs).**

## Objectives
* Develop a secure and efficient web-based automated grading system within the project timeline
* Increase student satisfaction with instant automated feedback
* Reduce faculty grading time by at least 50%
* Provide exportable reports compatible with external LMS platforms
* Ensure full system security through auditable authentication, authorization, and sandboxed code execution

## User Roles and Permissions
### Student
* Secure login and authentication
* Access assignments and starter code
* Submit and test code against public test cases
* Receive instant feedback
* View grading results and reports
* Participate in group assignments

### Faculty
* Create and manage classes
* Create assignments
* Define grading rubrics (weighted/unweighted)
* Manage group assignments
* Select grading mode (group or individual grading)
* Assign/remove teaching assistants
* Review submissions and reports

### Teaching Assistant(TA)
* Access assigned courses only
* Review and grade submissions
* Provide feedback
* Grade group assignments (based on faculty-defined grading mode)
* View plagiarism and AI-detection reports
* Access reporting dashboards (limited scope)
* Cannot create/modify/delete classes or assignments
* Cannot export institution-level analytics
* Export LMS-compatible reports
* Access plagiarism and AI-detection reports
